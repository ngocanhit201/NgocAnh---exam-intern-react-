# React + Vite Project

Guideline:
```
1. Fork this repository to your GitHub account
2. Clone the repository to your local machine
3. Start the development server locally:
    yarn dev
4. Make a web like video in /public/requirement.webm:
    - list products with responsive
    - can add product to cart
    - cart just need to show list items
    Note: do not use lib for any action, can use any lib css
5. After you finalize your code, push it and send us your repository link for feedback
```
